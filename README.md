# Лендинг Rapid REX

Промо-страница откосной косилки Rapid REX для B2B-аудитории — дорожных предприятий
и организаций, занимающихся содержанием дорог. Статичный одностраничный сайт без бэкенда.

## Запуск

Открой `index.html` в браузере — или запусти локальный сервер:

```bash
# Python
python3 -m http.server 8000
```

## Структура

```
index.html     — единственная страница
памятки/       — рабочие заметки
```
