# Лендинг Rapid REX

## О проекте
Лендинг откосной косилки Rapid REX. Аудитория — дорожные предприятия (B2B),
которые покупают промышленное оборудование. Цель страницы — представить модель
и вызвать интерес к покупке.

## Технологии
- HTML5 (один файл — index.html)
- Tailwind CSS v4 через CDN (jsdelivr)
- Шрифт Roboto Condensed — Google Fonts
- Иконки Lucide — unpkg
- Без фреймворков и бэкенда

## Структура проекта
```
лендинг-rapid-rex/
├── index.html     — единственная страница лендинга
├── памятки/       — долгосрочные факты о проекте
├── CLAUDE.md
├── README.md
└── .gitignore
```
